Scenariodataclass

A scenario represents a (task, data distribution). It is usually based on some raw dataset and is converted into a list ofInstance s. Override this class.

Note: the constructor should be lightweight,get_instances should do all the heavy lifting.

Extra metadata (e.g., whether this is a question answering or commonsense task)
